SoundPad is a button box designed for use with Leppsoft software of the same name. https://www.leppsoft.com/soundpad/en/

It features 20 buttons, each sends a unique key combination when pressed. A two position, latching switch(page switch) effectively doubles the number of buttons to 40. 
A footswitch can also be attached. Another two position latching switch changes the functionality of the footswitch, enabling two distinct foot switch modes.
